此版本带来了许多新功能，比如实现新的装饰器标准、更好地支持 Node 和打包工具中的 ESM 项目的功能、为库作者提供控制泛型推导 (generic inference) 的新方法、扩展 JSDoc 功能、简化配置，以及引入其他改进等

目标是让 TypeScript 更小、更简单、更快速
