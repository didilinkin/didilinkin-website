## 常用命令
```bash
# 重新加载配置
service nginx reload

# Nginx 重启
nginx -s reload
service nginx restart

# nginx 配置语法检查
nginx -t

# 查看错误日志
cat /var/log/nginx/error.log
```

### `resolver` 功能

:::info

当`error.log`提示 `no resolver defined to resolve`错误

说明 反向代理功能内部需要指定 DNS解析服务

添加一个 `可访问`的 DNS服务即可 (国外DNS服务 国内无法使用)

:::
