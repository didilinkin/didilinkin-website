## Mac 向 Linux 上传 文件

> 参考文章:
> [ssh 命令上传本地文件到 Linux 服务器](https://www.jianshu.com/p/c43105320695)

```bash
# 在 Mac 环境下
scp [文件本地的路径] [用户名@服务器地址:要放置的路径地址]

# 比如:
scp /Users/yanxiaodi/Downloads/caddy_v1.0.3_linux_amd64_custom.tar.gz root@47.105.184.229:/root/download

# 然后输入 密码即可
```

## Linux 向 Mac 下载文件

> 参考文章：
> Linux 压缩(打包)文件夹 tar/zip
>
> 参考文章：
> [将文件从 Linux 服务器复制到 Mac 桌面](https://qastack.cn/superuser/517943/copy-files-from-linux-server-to-mac-desktop)
>
> 参考文章：
> [Mac 与 Linux 之间传输文件](https://www.jianshu.com/p/fe31642e4f24)

```bash
# 首先使用 zip命令打包
# zip [参数] [打包后的文件名] [打包的目录路径]
zip -r ./xahot.zip .
```

#### Mac 本地操作

```bash
# sftp链接服务器
sftp root@47.105.184.229

# get [目标文件] [. - 下载到当前目录]
get /root/production/backup_html.zip .

# 解压
unzip [xahot.zip]
```

# Linux 在 Docker 中 复制容器内文件 到 指定文件夹 (备份)

> 参考文章：
>
> [Docker 文件目录和容器内部操作](https://www.cnblogs.com/merray/p/12849770.html)
>
> [docker 下 Gitlab 如何进行备份恢复与迁移？](https://www.cxyzjd.com/article/yaodunlin/96437668)
>
> [Docker 进入容器](https://yeasy.gitbook.io/docker_practice/container/attach_exec)
>
> Docker 文件夹地址：cd /var/lib/docker

```bash

# 首先进入 Docker容器内部
# docker exec -it [容器 ID] bash
docker exec -it c1d9 bash

# docker [容器 ID]:[容器内部文件路径] [目标文件夹]
docker cp c1d938d1c999:/var/opt/gitlab/backups /root/

```
