## use-http
> 可用来替代Fetch API

官网地址: [https://use-http.com/#/](https://use-http.com/#/)

## useMedia
> CSS媒体查询的方法

## Constate
> Constate是一个hook package, 可将本地状态提升到React Context

## Redux hooks
> Redux 全局状态管理器

## React hook form
> React hook form是一个与Formik和Redux表单相似的表单校验hook库

## useDebounce
> useDebounce 表示一个用于防抖的小 hook

## useLocalStorage
> 在 `localStorage` 中提取和设置数据

## usePortal
> usePortal 使创建下拉菜单，模态，通知弹出窗口，工具提示等变得非常容易

## useHover
> useHover是一个React state hook, 它确定是否正在hover React元素

## React router hooks
> 路由和获取应用程序URL历史记录等

主要功能是:

* useHistory
* useLocation
* useParams
* useRouteMatch

## react-use
> react-use是一个必不可少的 React Hooks集合

## ahooks
> 阿里的 Hooks集合
