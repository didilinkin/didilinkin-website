<!--truncate-->

<br />

## 简要介绍:

* [GitHub Actions](https://github.com/features/actions) 是 GitHub 的持续集成服务, 于2018年10月推出

* [Github Pages](https://docs.github.com/en/pages) 是一个可以从你的Github 源码仓库中直接生成 项目页面的静态站点托管服务

* [Next.js](https://nextjs.org/) 是一个用于构建服务器渲染 React 应用的框架

### 在本教程中的步骤:

1. 将本地开发好的 `Next.js`项目上传到 GitHub, 启用 `GitHub Pages`

2. 配置 `GitHub Actions` workflow, 以便在 push时自动构建、导出和部署 静态网站到 `GitHub Pages`

3. 修改 `Next.js`的配置选项以适应 `GitHub Actions` URL结构
